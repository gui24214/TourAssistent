# TourAssistant

## Descrição do Projeto

O TourAssistant é um sistema concebido para apoiar a exploração de espaços físicos através de uma aplicação digital que funciona como um assistente de visita.

A ideia do projeto é permitir que um utilizador explore um determinado local — como um museu, exposição, espaço histórico — utilizando o seu próprio dispositivo móvel para aceder a informações, conteúdos multimédia e orientação durante a visita.

O sistema procura melhorar a forma como as pessoas descobrem e compreendem os espaços que visitam, fornecendo conteúdos digitais associados aos diferentes pontos de interesse existentes nesses locais.



## Conceito do Sistema do TourAssistant


O TourAssistant foi pensado para ser reutilizável em diferentes contextos, não estando limitado a um único tipo de espaço.

Para tornar o sistema mais flexível, os locais são representados através de uma estrutura composta por três elementos principais:

**Entidade**
Representa o local ou organização principal que disponibiliza a experiência de visita (por exemplo um museu, parque, centro histórico ou exposição).

**Áreas**
Divisões ou zonas dentro da entidade que ajudam a organizar o espaço.

**Itens**
Pontos de interesse específicos que podem conter conteúdos informativos associados.

Esta estrutura permite organizar qualquer espaço visitável de forma clara e adaptar o sistema a diferentes tipos de ambientes.

---

## Objetivo do Projeto

O principal objetivo do TourAssistant é criar uma plataforma que permita apresentar conteúdos digitais associados a espaços físicos, ajudando os visitantes a compreender melhor os locais que estão a explorar.

A aplicação funciona como um **assistente de visita**, fornecendo informação contextual sobre os diferentes pontos de interesse existentes numa determinada entidade.

---

## Estrutura Geral da Solução

O projeto é composto por dois componentes principais:

**Aplicação de visita**
Responsável pela interação com o utilizador e pela apresentação dos conteúdos durante a visita.

**Sistema de gestão de conteúdos**
Responsável por armazenar e organizar a informação associada às entidades, áreas e itens que fazem parte do sistema.

Esta separação permite que os conteúdos possam ser geridos de forma independente da aplicação.

---

## Estado do Projeto

O TourAssistant encontra-se atualmente em fase de desenvolvimento.
Nesta fase o foco está na definição da estrutura base do sistema e na criação da arquitetura que permitirá suportar diferentes tipos de experiências de visita.
